Enhancing File Sharing and Data Management Capabilities at Batangas State University - Alangilan Faculty Through a Distributed File System

Abstract

The study aims to enhance file sharing and data management capabilities at Batangas State University - Alangilan Faculty through a Distributed File System (DFS). The project involves in creating a logical design for file distribution across shared folders and mapping the virtual DFS path to the physical location of files using Django Framework. The web application acts as a gateway for accessing files in the DFS, providing the user file management, shared file management, search function, and admin tracking. The DFS Namespace configuration allows the web application to store files and access multiple storage. Furthermore, a database allows the web application administrator to track details, including users, file details, and shared files details. The project successfully achieved its research objectives and provided a robust platform for file handling. The Django framework maps the DFS paths to their physical locations, demonstrating the practical benefits of combining distributed computing with web technologies. The design project sets a precedent for future endeavors in distributed file systems and web application integration, offering a scalable methodology for data management. This innovative method creates new opportunities for software development and data processing innovation while also enhancing the capabilities of existing technologies.

Faculty Staff

Faculty Staff Login

To login, users should use their credentials. Press the “Login” button to proceed into the app, as shown in the Figure above. (Note: Only admins will be able to register an account for users to access the web app. This ensures that the website is accessible only to the faculty staff)

Uploading of Files

When users enter the website, they will see the 'Upload Files' button located in the upper left corner of the screen, shown in the Figure above. Press the ‘Upload Files’ button and a new window will prompt to appear on the screen, allowing users to select files.

Selecting of Files

After the new window appears, users can select the file they want to upload within the web app. Once the file is selected, users should press the 'Open' button, as shown in the Figure above, to upload it into the web app.

Sharing of Files

When the file is uploaded, users will be able to share it with others. Shown in the Figure above, users must click on the 'Share' button. A pop-up window will then appear, prompting the user to enter the email address of the recipient. (Note: Files will only be shared with approved recipients to prevent them from spreading elsewhere)

Downloading of Files

Once the file is shared to the recipient, the recipient will be able to download the file. Navigate the 'Shared Files' category, which is located on the left side of the screen, as shown in the Figure above. Within this category, recipients will find the files shared with their account. Press the ‘Download’ button to the file that needs to be downloaded, this will allow the recipient to have access to the shared file.

Searching of Files

Simply click on the 'Search Tab', positioned at the upper center of the screen, as depicted in the Figure above. This action will open a search bar. Then, enter the keywords or file names to easily locate the desired file.

Deleting of Files

Press the 'Delete' button, located in the middle of the 'Download' and 'Share' buttons, as shown in the Figure above, on the desired file to initiate its deletion. After confirming the deletion prompt, the file will be permanently removed from the system. Ensure to double-check before proceeding, as this action cannot be undone.

Logging out of Account for Faculty Staff

To log out, click the 'Logout' button located on the upper right corner of the screen, as illustrated in the Figure above. Upon clicking the button, the system will redirect back to the login section, ensuring a secure logout process. After logging out, it's advisable to close the browser window for added security. This helps prevent unauthorized access to users' accounts in case the device is shared or left unattended.

Administrator

Admin Login

To login, the admin must use their credentials. Press the ‘Login’ button to proceed into the app, as shown in the Figure above. (Note: Only authorized admins can log in to this section, ensuring that any faculty staff members cannot access it)

Adding of User Account

In the 'Add User' category, which is located on the left side of the screen as shown in the Figure above, fill out the form using the new user's credentials for registration. Once completed, press the 'Add Account' button, and a new user will be successfully registered.

Managing of Users Accounts

Managing accounts can only be done by the admin. The 'Manage Account' category can be seen on the left side of the screen, under the 'Add User' category, as shown in the Figure above. To edit the account of an existing user, click the 'Edit User Account' button located on the left side of the 'Change Password Account' button. Once clicked, it will immediately navigate to the 'Edit Account' section. Admins will fill out the necessary forms according to the required information. Once done, press the 'Save Edit Account' button. (2) To change the password of any existing user, click the 'Change Password Account' button, positioned between the 'Edit Account' and 'Delete' buttons. Admins will then be directed to the 'Change Password Account' section. Fill out any required information, then input the new password for the existing user. Once the new password is entered, press the 'Save Edit Button'. (3) To delete any existing user account, simply click the 'Delete Account' button, located on the right side of the 'Change Password Account' button. The selected user's account will be deleted upon clicking.

Monitoring of Available Storage

To monitor the availability of storage on each device, select the 'Available Storage' category located on the left side of the screen, under the 'Manage Account' category, as illustrated in the Figure above. Upon selection, the admin will be able to monitor the drive storage of each device and its overall usage.

Tracking of File Details

To track files, navigate to the 'Track File Details' category, located on the left side of the screen, above the 'Track Share Files' category, as illustrated in the Figure above. The 'File Details' section will be displayed upon selecting this category. Press the 'File Details' button to view the details of the file.

Tracking of Shared Files

To track shared files, navigate to the 'Track Shared Files' category, that can be found on the left side of the screen, below the 'Track File Details' category, as shown in the Figure above. The 'Shared Files Details' section will be displayed upon selecting this category. Press the 'File Detail' button to view the details of the shared file.

Logging out of Account for Admin

To initiate logout, simply select the 'Logout' button that can be found in the lower left corner of the screen, under the ‘Track Share Files’ category, as demonstrated in the Figure above. Upon pressing, the system will promptly return to the login section, guaranteeing a secure logout procedure. After completing the logout process, it's recommended to close the browser window to enhance security, reducing the risk of unauthorized access to admin accounts, especially on shared or unattended devices.
