Enhancing File Sharing and Data Management Capabilities at Batangas State University - Alangilan Faculty Through a Distributed File System

Abstract

The study aims to enhance file sharing and data management capabilities at Batangas State University - Alangilan Faculty through a Distributed File System (DFS). The project involves in creating a logical design for file distribution across shared folders and mapping the virtual DFS path to the physical location of files using Django Framework. The web application acts as a gateway for accessing files in the DFS, providing the user file management, shared file management, search function, and admin tracking. The DFS Namespace configuration allows the web application to store files and access multiple storage. Furthermore, a database allows the web application administrator to track details, including users, file details, and shared files details. The project successfully achieved its research objectives and provided a robust platform for file handling. The Django framework maps the DFS paths to their physical locations, demonstrating the practical benefits of combining distributed computing with web technologies. The design project sets a precedent for future endeavors in distributed file systems and web application integration, offering a scalable methodology for data management. This innovative method creates new opportunities for software development and data processing innovation while also enhancing the capabilities of existing technologies.

Faculty Staff Login

To login, users should use their credentials. Press the “Login” button to proceed into the app, as shown in Figure A.1. (Note: Only admins will be able to register an account for users to access the web app. This ensures that the website is accessible only to the faculty staff)

Uploading of Files

When users enter the website, they will see the 'Upload Files' button located in the upper left corner of the screen, shown in Figure A.2. Press the ‘Upload Files’ button and a new window will prompt to appear on the screen, allowing users to select files.

Selecting of Files

After the new window appears, users can select the file they want to upload within the web app. Once the file is selected, users should press the 'Open' button, as shown in Figure A.3, to upload it into the web app.

Sharing of Files

When the file is uploaded, users will be able to share it with others. Shown in Figure A.4, users must click on the 'Share' button. A pop-up window will then appear, prompting the user to enter the email address of the recipient. (Note: Files will only be shared with approved recipients to prevent them from spreading elsewhere)
